Scripts

In order for mount off to work properly, the script containing every timestamp for every event should be handwritten.
Currently supported events are:

  • PLAYER: number of remaining players has been updated
  • MOUNT: a new mount has been selected

See the following example:

...
{"time": "0:00", "event": "PLAYER", "players": 39},
{"time": "0:51", "event": "MOUNT", "mount": "Snowy Gryphon"},
...

Full configuration of scripts can be found on github.
