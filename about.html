About

This website offers a new way interactive to enjoy Asmongold's Mount offs.
To start, hit Play and select one of the available episodes.
All unprocessed episodes are available at Archive.
Note that it is currently in early stage of development.

Currently available features:

  • Tracking current mount
  • Tracking remaining players
  • Scripts written for 4/37 Mount Offs

Incoming features:

  • Integration with WoW Profile API - Track your own progress against Mount Offs
  • Mount off data visualization and statistics
  • Pretty UI

If you have any additional feature suggestions, hit me up on twitter.
