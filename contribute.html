Contribute

There are several ways you can contribute to this project at this point:
  1. The most valuable contribution would be providing recordings of currently unavailable Mount Offs.
    Currently archive lists 37 of them, but some are missing.
  2. The scripts for each mount off needs to be handwritten and it's pretty tedious work. If you wish to help with this, you can read the details here.
  3. Spread the word!
